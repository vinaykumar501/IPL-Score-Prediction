# 🏏 IPL Score Prediction Web App

This project predicts the **final score of an IPL match** based on the current match situation using **Machine Learning**.
The application is built using **Python, Scikit-learn, and Streamlit** to provide an interactive web interface.

Users can enter match details such as batting team, bowling team, current score, balls left, wickets left, and run rate to predict the expected final score.

# 🚀 Demo

The web application allows users to:

1. Select the **batting team**
2. Select the **bowling team**
3. Enter the **current score**
4. Enter the **balls left**
5. Enter the **wickets left**
6. Enter the **current run rate**

After clicking **Predict Score**, the model predicts the **final total score**.

# 🔧 Installation

Clone the repository:

```
git clone https://github.com/yourusername/IPL-Score-Prediction.git
```

Navigate to the project folder:

```
cd IPL-Score-Prediction
```

Install required dependencies:

```
pip install -r requirements.txt
```

---

# ▶️ Run the Application

Start the Streamlit web app using:

```
streamlit run app.py
```

The application will open automatically in your browser.
